# 竞赛咨询知识库种子

这个目录放适合导入竞赛咨询知识库的 Markdown 样例和模板。它们不是完整知识库，而是建议的资料形态。

推荐知识库名称：

```bash
deeptutor kb create competition-consulting --doc deeptutor/agents/competition_consulting/knowledge_seed/math_competition_gold_path.md
deeptutor kb create competition-consulting --doc deeptutor/agents/competition_consulting/knowledge_seed/learning_experience_template.md
```

如果 CLI 对同名知识库不支持重复 create，后续应使用项目已有的追加文档命令或知识库页面上传。

资料编写原则：

- 标明资料类型：访谈、复盘、路线图、书单、题单、家长沟通。
- 标明适用阶段：入门、基础、专题强化、模拟赛、选拔、赛前。
- 区分事实和建议：真实个案必须有来源，通用经验要写明是归纳。
- 不要虚构真实学生、学校、奖项或年份。
- 每篇材料结尾加“可迁移做法”和“不适用情况”。
